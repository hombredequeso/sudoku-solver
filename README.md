# Sudoku Solver

sudoku-solver is a Typescript project for solving sudoku puzzles by brute force.

## Installation

Use the [yarn](https://yarnpkg.com/) package manager.


```bash
yarn
```

## Usage

Currently, only tests for solving up to a 9x9 grid is implemented.

```bash
yarn test
```
